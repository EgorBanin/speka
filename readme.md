# Speka — спецификация API для тупых :-)

При унифицированном формате API удобно не описывать всю спеку в каком-нибудь OpenAPI, а бахнуть только самое главное: различающиеся части запросов и ответов, коды ошибок. При этом типы можно вывести из значений, сделав спеку более наглядной. Speka — попытка реализовать эту идею.

```
go install github.com/egorbanin/speka@latest
```
